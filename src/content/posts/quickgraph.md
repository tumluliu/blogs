##### QuickGraph 3.0: Generic Graph Data Structures and Algorithms for .Net.

**QuickGraph 3.0** provides *generic* directed/undirected graph datastructures and algorithms for .Net 2.0 and up. QuickGraph comes with algorithms such as depth first seach, breath first search, A\* search, shortest path, k-shortest path, maximum flow, minimum spanning tree, least common ancestors, etc... QuickGraph supports [MSAGL](http://www.codeplex.com/quickgraph/Wiki/View.aspx?title=MSAGL&referringTitle=Home), [GLEE](http://www.codeplex.com/quickgraph/Wiki/View.aspx?title=GLEE&referringTitle=Home), and [Graphviz](http://www.codeplex.com/quickgraph/Wiki/View.aspx?title=Graphviz&referringTitle=Home) to render the graphs, serialization to [GraphML](http://www.codeplex.com/quickgraph/Wiki/View.aspx?title=GraphML&referringTitle=Home), etc...

- **QuickGraph 3.0** supports a .NET 2.0 build, and a more fancy .NET 3.5 build with extension methods support.
